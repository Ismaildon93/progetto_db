// APPLICATIVO PHP

1) L'appicativo php si trova nella cartella applicativo_php_cinguettio
ed è strutturato nel seguente modo

	index.php --> file di index
	config --> cartella contenente i file di configurazione
	css --> cartella contenente i fogli di stile
	function --> cartella contenente i file delle funzioni utilizzate per il sito
	img --> cartella contenente le immagini utilizzate nel sito
	|
	 --> img/icone --> cartella contenente le icone del sito
	 --> img/utenti--> cartella contente le immagini degli utenti
		|
		 --> img/utenti/profilo --> cartella contenente le foto di profilo degli utenti
		 --> img/utenti/profilo --> cartella contente le foto caricate dalla bacheca 
									(*se si vuole fare una prova di caricamento di un cinguettio di tipo foto
									si possono prendere le immagini da qui o dalla cartella profilo degli utenti*)
	javascript --> cartella contente i file javascript
	query --> cartella contente le funzioni per le query
	validazione --> cartella contenente gli script per la validazione dell'input inserito dall utente
	visual --> cartella contente gli script php per la visualizzazione delle pagine
	
// FILE SQL 

2) I file sql vanno eseguiti nel seguente ordine

	1) db_creazione_tabelle -> contiene gli script per la creazione delle tabelle 
	2) db_trigger -> contiene gli script per l'esecuzione dei trigger
	3) popolamento_database -> contiene gli script per il popolamento del database


// DOCUMENTAZIONE

	1) documentazione_cinguettio --> contiene la documentazione del progetto 
	2) query_sql_e_trigger --> contiene la risoluzione delle query sql e i trigger con i vincoli risolti
	3) vincoli_non_implementabili_in_mysql --> il nome è autoesplicativo
